Privacy Policy

Effective: 15 July 2026 · Applies to the FastPen Windows application and the fastpen.app website

In short: FastPen processes your voice entirely on your own computer. Your audio and your dictated text never leave your device. The app has no accounts, no telemetry, no analytics, and no ads. We do not collect, store, sell or share any personal data — we have no servers that could receive it.

What the app processes

  • Microphone audio. While recording is active (you control it with a hotkey), audio is processed in memory on your device to produce text. Audio is not saved to disk and is never transmitted anywhere, with one exception: if you explicitly enable the diagnostic audio dump (an advanced troubleshooting option, off by default), the audio segments are saved as local .wav files in the app's data folder, which you can inspect and delete at any time. They never leave your machine.
  • Dictated text. The recognized text is typed into the application you choose. In the default "paste" input mode the text passes briefly through the Windows clipboard. FastPen does not log or store your dictated text, with one exception: if you explicitly enable the diagnostic log in Settings, recent transcripts are written to a local text file (fastpen_debug.log) that you can inspect and delete at any time. It never leaves your machine.
  • Settings. Your preferences (hotkey, model choice, custom words, etc.) are stored in local files on your device. The Microsoft Store version stores them in your user app data, which Windows removes when you uninstall the app.

Network connections the app makes

FastPen works offline. It makes network requests only in these cases:

  • Speech model download (first run, or when you select a new model): the model files are fetched from public repositories (GitHub, Hugging Face). No personal data is sent — it is an ordinary file download.
  • Optional AI cleanup: if you enable it, FastPen talks to an Ollama server running locally on your computer (127.0.0.1) — this traffic never leaves your machine. In the website (portable) edition, FastPen can download and install Ollama for you after you explicitly agree; the Microsoft Store edition never installs third-party software (you install Ollama yourself if you want AI cleanup). When Ollama first downloads its language model, that is again a public file download by Ollama.

There is no telemetry, no crash reporting, no update pinging, and no analytics of any kind in the app. You can verify this: block FastPen with a firewall after the model download — it keeps working.

The website and donations

  • The fastpen.app website is a static site hosted on GitHub Pages. We run no analytics on it. GitHub, as the hosting provider, may log standard technical data (such as IP addresses) — see GitHub's privacy statement.
  • Donations are handled by Ko-fi and its payment provider (Stripe) on their own sites, under their own privacy policies. FastPen (the app) knows nothing about them, and we receive no card details.

Children

FastPen collects no data from anyone, including children.

Changes

If we ever change how FastPen handles data, we will update this page and note the change in the release notes before the new version ships.
