# FastPen

**Fast, private, on-device dictation for Windows.**

Press a hotkey, speak, and your words are typed into any app — email, Word,
browser, chat. Speech recognition runs 100% on your computer: no cloud, no
account, no character limits.

- 🌍 25 languages with automatic detection (English, German, French, Spanish, Hungarian, …)
- ⚡ Near real-time: text appears the moment you pause
- 🔒 Private by design: works fully offline after the first model download
- 📦 9 MB download; the speech model (~465 MB) is fetched automatically on first run

## Download
